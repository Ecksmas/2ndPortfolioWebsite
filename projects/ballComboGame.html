A game that was created with the purpose of solidifiying different skills learning via different game development courses which mainly cover the GoDot engine. The game in it's current state is fully functional and has the possibitliy to save the score and retrive it upon loading the game again.

During the project I learned a lot of what it takes to create a simple arcade game and what effect different aspects of a game bring to the table. Some of these aspects were, adding sound effects, music and creating a original character.
